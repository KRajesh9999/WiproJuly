# Program Structure
A detailed breakdown of modules and daily learning for the .NET Full Stack Course:

### **1. Introduction to .NET, C#, OOP & Git**
- **Date:** 21 July 2025
- **Topics:**
  - Getting started with Git basics
  - Version Control: centralized vs distributed systems
  - CI/CD concepts and tools (Jenkins, GitHub Actions, GitLab CI)
  - Introduction to .NET Framework
    - Architecture, Common type system & language runtime
    - Base class library, Types in C#
    - Classes, Console I/O
    - Assemblies, Value/Reference types, Boxing/Unboxing, Strings & StringBuilder, Nullable types

***

### **2. Core C# Programming**

- **Date:** 22 July 2025
- **Topics:**
  - Arrays (Single, Multi-dimensional, Jagged)
  - Looping Constructs:
    - If, For, Foreach, Switch, While, Do-while

***

### **3. Properties, Functions and Access Modifiers**

- **Date:** 23 July 2025
- **Topics:**
  - Access modifiers (Private, Public, Protected, Internal)

  - Namespaces
  - Unit Testing basics

***

### **4. Inheritance & OOP Principles in C#.NET**

- **Date:** 24 July 2025
- **Topics:**
  - Types of Inheritance (Single, Multilevel, Interface-based, Hierarchical, Hybrid)
  - Polymorphism (Override, Overload)
  - Encapsulation & Abstraction
  - Properties, Static classes/methods, Structs, Interface, Abstract, Enumeration
  - Exception Handling overview
***

### **5. File Handling & Collections**

- **Date:** 25 July 2025
- **Topics:**
  - File I/O: Create, Write, Append, Delete, Copy

### **6. Advanced C#, Exception Handling & Sorting Algorithms**

- **Date:** 26 July 2025

  - Indexers & Attributes

### **7. Delegates, Reflection, Generics & Threading**

- **Date:** 28 July 2025

  - Delegates (Single/Multi-cast, Lambda, Func, Action, Predicate, Events)

### **8. Unit Testing & Test-Driven Development**
